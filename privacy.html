Privacy Policy

DesiMod AI respects user privacy and follows Reddit platform rules.

1. Data Collection

We only process Reddit posts and comments for moderation analysis. No personal user data is stored permanently.

2. Data Usage

Content is sent to AI (Google Gemini API) only for classification (toxic, scam, spam detection).

3. Data Storage

We store only moderation metadata such as strike counts and action logs.

4. Third-Party Services

This app uses Google Gemini AI for content classification.

5. Security

All data is processed securely and only accessible to subreddit moderators.

6. Contact

For concerns, contact the developer via Reddit.
